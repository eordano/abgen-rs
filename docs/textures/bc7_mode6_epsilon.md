# BC7 mode-6 preference epsilon: an inert tiebreak, and the real residual

**Why it matters:** abgen-rs appeared to under-pick BC7 mode 6 (1-subset RGBA) relative to the reference, which would diverge the mode bits and payload on many blocks. The original diagnosis pointed at mode 6 being lost to 2-subset modes on smooth gradients. This note records that the diagnosis was stale and identifies the actual residual, so the parity work is not aimed at the wrong target.

**How it works:** A small asymmetric bias was added: non-mode-6 candidate errors are inflated by a tiny fraction before the per-mode comparison, so exact ties tip toward mode 6. In practice this is inert. The under-pick that motivated it was a measurement against an outdated binary; the current encoder already matches the no-bias result, and the bias only changes the outcome on very high-error blocks that barely occur in any tested corpus. The chosen shift is the largest (weakest) value compatible with the parity ceiling — anything stronger flips low-margin mode-1 blocks into mode 6 and regresses a fixture. The machinery is kept in place as a conservative, no-op option for the future, not as an active fix.

The genuine residual is the opposite of the original framing, and this is the negative finding: abgen-rs over-picks mode 5, not under-picks mode 6. On smooth 1D gradients the reference picks mode 6 where we pick mode 5; on 2D gradients the reference picks a 2-subset mode where we again pick mode 5. Mode 5 wins on raw error because its coarse index levels happen to line up with a small per-block gradient range, and its rotation lets it spend the alpha slot on the highest-variance color channel at higher endpoint precision. A multiplicative bias cannot close this: the error margins favoring mode 5 are orders of magnitude larger than any tie-break epsilon that stays within the parity ceiling. Closing it requires a structural rule — a "rotation worth it" gate that only considers mode-5 rotation when alpha variance is significant, or disabling mode 5 on constant-alpha low-variance blocks so the contest falls to modes 1/2/3/6 — each needing its own corpus sweep.
