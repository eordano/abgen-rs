# RecalculateTangents: degenerate gate on the projection, branch-dependent handedness

**Why it matters:** When a glTF primitive has normals but no TANGENT accessor, the importer runs `Mesh.RecalculateTangents()` and abgen reproduces it in `src/tangents.rs`. On degenerate seam vertices (near-zero accumulated tangent, mirrored/zero-area UV triangles) abgen's previous final stage disagreed with the reference in two ways: the degenerate test was applied to the wrong quantity, and the tangent `w` handedness consulted the accumulated binormal residue even when the tangent itself had been replaced by the axis fallback.

**How it was derived:** `examples/tangprobe` dumps the recompute inputs (f32 positions/normals/uvs, index buffer, abgen's tangents) for every tangent-bearing primitive, and a full-pipeline simulation was scored for bit-exactness against the reference tangent lanes across the affected meshes (a seam-heavy wearable and an axis-aligned-floor scene glb that bracket the thresholds from both sides):

- **Gate:** the epsilon test is on the magnitude of the Gram-Schmidt *projection* (`tan1` minus its normal component), not on `|tan1|`. A seam vertex with projection magnitude `8.6e-7` (but `|tan1| = 1.7e-6`) takes the axis fallback in the converter reference — its xyz matches the fallback-axis projection formula exactly — while floor vertices with projection magnitudes down to `5.6e-6` keep the projected direction. The boundary is consistent with the same `1e-6` epsilon the old code applied to `|tan1|`. Since the projection magnitude never exceeds `|tan1|`, the corrected gate strictly widens the fallback set.
- **Handedness:** a vertex that took the axis fallback decides `w` from the second-axis dot directly, ignoring the tiny accumulated binormal. Reference-corpus seam vertices carry `w = +1` where the raw residue dot is negative, while non-degenerate vertices with comparably tiny binormals (down to `9.7e-6` on axis-aligned floors) follow the raw dot's sign — so the discriminator is the branch, not a magnitude threshold on the binormal.

**Where:** `src/tangents.rs`, final per-vertex stage. Bulk 1-ULP noise in the accumulated lanes (float-order differences against the converter's native recompute pass) is unaffected and remains the residual on these meshes.
